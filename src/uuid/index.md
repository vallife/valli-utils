## uuid

Demo:

```tsx | pure
import { uuid } from '@vallife/utils-valli';

const id1 = uuid(8);

console.log(id1); // -> BUfS1qrn

const id2 = uuid(10);
console.log(id2); // -> Zd9hoanTYu

const id3 = uuid(16);
console.log(id3); // -> tXIEXMFXG77P4dN8
```
